# PSKiller
Easy to manage ps using a GUI.
The tool is based on PyQt4.

# Requirements
- python 2.7
- PyQt4
- psutil
- apscheduler

## Supported Platforms

- Linux

- OS X

# Installation
You must [install pyqt4](http://pyqt.sourceforge.net/Docs/PyQt4/installation.html) on your OS.

# How to use
```sh
python ps_killer.py
```

Enjoy yourself.

Any contributions on this welcome.
